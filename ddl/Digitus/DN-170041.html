About this device

I tested the Digitus DN-170040 and DN-170041 with the blazer_usb driver and it works as expected. The same model was also sold as Voltronic Power GalleonX9-RT LCD-1-3K.

It is connected through USB and is recognized as:

Bus 001 Device 002: ID 0665:5161 Cypress Semiconductor USB to Serial

See also: DigitusDN-170041nutdrv_qx2.7.401.dev

Reported at:

  • https://github.com/networkupstools/nut/issues/1251

Vendor/Manufacturer information:

  • http://ftp.assmann.com/pub/DN-/DN-170041___4016032365907

Further system data or configuration example:

Driver configuration
[digitus]
    driver = blazer_usb
    port = auto
    vendorid = 0665
    productid = 5161
    subdriver = cypress
    desc = "Digitus DN-170041"
