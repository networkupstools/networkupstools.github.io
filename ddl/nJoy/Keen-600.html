About this device

From https://github.com/networkupstools/nut/issues/867 :

  • Manufacturer: nJoy

  • Device: Keen 600 USB

  • Website: https://www.njoy.ro/UPS/keen-600

    • No separate site for USB version.

It doesn’t report battery.charge nor battery.runtime

Shutdown sequence working, as I set AT ONBATT * START-TIMER shutdown 30 in upssched.conf. I guess AT LOWBATT won’t work without reported charge.

The usb-serial adapter is built into the UPS. I didn’t install anything extra manually, the driver is built into the kernel.

In this report, we see upsc output when the device is on main power.

See also nJoy__Keen-600__blazer_ser__2.7.4__02.dev (on battery) and nJoy__Keen-600__nutdrv_qx__2.7.4__03.dev (another driver)

  • https://github.com/networkupstools/nut/issues/867
